Mathematical Background

A = U Σ VT

SVD Decomposition: Any matrix A can be decomposed into three matrices:

  • U: Left singular vectors (orthogonal matrix)
  • Σ: Singular values (diagonal matrix, σ₁ ≥ σ₂ ≥ ... ≥ σᵣ)
  • VT: Right singular vectors transposed (orthogonal matrix)

Reconstruction: A ≈ Uₖ Σₖ VₖT, where k is the number of components used.

How SVD Works

Step 1: Data Centering

Subtract the mean from each feature to center the data around zero.

Step 2: SVD Decomposition

Decompose the centered matrix into U, Σ, and VT matrices.

Step 3: Component Selection

Choose the top k components based on singular values (importance).

Step 4: Reconstruction

Reconstruct the original data using only the selected components.
